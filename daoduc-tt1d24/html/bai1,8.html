b) Danh sách định nghĩa

Pixel
Viết tắt của phần tử ảnh. Pixel là những điểm ảnh nhỏ tạo nên hình ảnh hiển thị trên màn hình.
Độ phân giải
Chất lượng hiển thị của màn hình. Độ phân giải càng cao thì hình ảnh càng sắc nét.
Trình duyệt web
Phần mềm cho phép người dùng truy cập và hiển thị các trang web trên Internet.

Kiến thức nền tảng về Thiết kế Web

  1. HTML (HyperText Markup Language)
    1. Cấu trúc cơ bản của tài liệu HTML
    2. Các thẻ phổ biến: <head>, <body>, <h1>-<h6>, <p>, <a>, <img>
  2. CSS (Cascading Style Sheets)
    1. Khái niệm và vai trò của CSS
    2. Các cách nhúng CSS: nội tuyến, nội bộ, liên kết ngoài
    3. Thuộc tính định dạng văn bản, màu sắc, bố cục
  3. JavaScript
    1. Tạo tương tác động cho trang web
      1. Xử lý sự kiện người dùng: onclick, onchange...
      2. Thao tác với DOM: getElementById, innerHTML
    2. Biến, hàm và cấu trúc điều khiển
  4. Thực hành tổng hợp
  5. Tổng kết và hướng phát triển
